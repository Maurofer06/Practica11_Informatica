# Practica11_Informatica
Secuencia de encendido y apagado de dos LEDs de forma secuencial 

Esta practica funciona de forma que al darle corriente al arduino y a la placa de pruebas, segun la programaci√≥n, se encendera el LED numero 1 durante 10 segundos, al pasar este tiempo se pagara. A los 4 segundos de haberse apagado, se encendera el LED 2 durante 10 segundos y al pasar este tiempo se apagara. A los 4 segundos de que el LED 2 se quede apagado se repetira todo el ciclo de nuevo.
